# 06400 升降機裝修丙級－電氣配線模擬器

這是一個以純前端製作的互動式配線練習工具，主要用於模擬台灣「06400 升降機裝修丙級」術科測驗中的電氣配線作業。

專案目標是在手機與電腦瀏覽器中，提供容易操作的虛擬接線板，讓使用者可以認識元件、練習配線邏輯、操作開關及測試電驛狀態。

## 目前功能

- 支援電氣配線 A～D 題型
  - 優先選擇電路 A
  - 最新輸入優先電路 B
  - 電驛作動數量檢出電路 C
  - 二進位轉十進位解碼電路 D
- 練習模式
  - 題目提示
  - 逐步參考接線
  - 即時短路提醒
- 模擬測驗模式
  - 50 分鐘倒數計時
  - 完成後統一評分
- 自由配線模式
- 點擊兩個端子建立導線
- 支援多種導線顏色
- 導線刪除、清除及復原
- NFB 無熔絲開關操作
- 輸入按鈕操作
- X1～X4 電磁電驛吸合狀態模擬
- 元件用途與端子編號說明
- 建議練習步驟
- 使用瀏覽器 `localStorage` 自動保存進度
- 響應式介面，可在手機及電腦使用

## 使用方式

本專案不需要安裝套件或建置工具。

直接使用瀏覽器開啟 `index.html` 即可開始操作。

### 基本接線

1. 選擇導線顏色。
2. 點擊第一個端子作為起點。
3. 點擊第二個端子完成接線。
4. 點擊既有導線可以將它刪除。
5. 完成配線後將 NFB 切換至 `ON`。
6. 按住輸入按鈕，觀察 X1～X4 電驛狀態。
7. 使用「檢查結果」查看配線評分。

手機畫面可使用工作板下方的按鈕進行縮放及重設視圖。

## 元件

- `POWER`：控制電源 R、T
- `TB1`：控制電源端子台
- `NFB`：無熔絲開關
- `X1～X4`：電磁電驛
  - `A1`、`A2`：線圈端子
  - `C`：接點共用端
  - `NO`：常開接點
  - `NC`：常閉接點
- `輸入 1～4`：模擬輸入信號的按鈕
- `TB2`：電驛狀態及輸出檢測端子

## 模擬範圍

目前只處理術科中的「電氣配線作業」，不包含：

- 主鋼索製作
- 印刷電路板組立與焊接
- 剝線、壓接及端子鎖付
- 實際電壓、電流、電阻及故障物理模擬

本工具主要用於電路邏輯與端子位置練習，不能取代實體配線訓練、專業教師指導或正式技能檢定資料。
